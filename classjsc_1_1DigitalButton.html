◆ invertInputLogic()

void DigitalButton::invertInputLogic ( )

Invert input logic.

This function allows the user to dynamically change the logic while the program is running. Inverted logic means that if the input voltage level is high (e.g. 5 V DC), it will be treated as logic 0. Therefore, a low voltage level (0 V DC) will be represented as log 1.
